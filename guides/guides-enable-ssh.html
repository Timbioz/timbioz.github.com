Установка и настройка SSH доступа в Ubuntu server (16.10)

Сервер OpenSSH - представляет мощную коллекцию инструментов для удаленного контроля и обмена данными с сетевыми компьютерами

Установка

Для установки серверного приложения OpenSSH и связанных файлов используйте такую команду в терминале:

sudo apt-get install openssh-server

Конфигурация

Вы можете настроить поведение по умолчанию серверного приложения OpenSSH, sshd, редактируя файл /etc/ssh/sshd_config

Для доступа пользователя root закомментируте строку:

PermitRootLogin without-password

и добавьте ниже строку

PermitRootLogin yes

далее, перезапустите сервер SSH

service ssh restart

Теперь вы можете заходить через SSH используя данные root пользователя
